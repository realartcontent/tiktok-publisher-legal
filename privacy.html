Politica de Privacidade

Ultima atualizacao: 22 de junho de 2026.

Este Aplicativo e de uso pessoal/privado, operado por um unico desenvolvedor para gerenciar a propria conta do TikTok. Esta pagina explica quais dados sao tratados e como.

1. Dados coletados

  • Tokens de acesso/atualizacao do TikTok (OAuth2): obtidos apos o operador autorizar o Aplicativo via fluxo oficial do TikTok. Armazenados localmente, em texto, apenas na maquina do operador, e nunca transmitidos a terceiros.
  • Arquivos de video: fornecidos pelo proprio operador a partir de um diretorio local, usados unicamente para gerar legenda/hashtags (via Google Gemini API) e para upload na API oficial do TikTok.
  • Informacoes basicas de perfil TikTok (open id, nome de exibicao, avatar): obtidas via escopo user.info.basic apenas para identificar a conta autenticada.

2. Como os dados sao usados

Os dados sao usados exclusivamente para permitir que o Aplicativo gere propostas de legenda e hashtags com a API do Google Gemini, e para publicar/agendar videos na conta TikTok do proprio operador, mediante aprovacao manual antes de cada publicacao.

3. Compartilhamento de dados

Nenhum dado e vendido, alugado ou compartilhado com terceiros. Os unicos servicos externos contatados pelo Aplicativo sao: (a) a API oficial do TikTok, para autenticacao e publicacao; e (b) a API do Google Gemini, para analise do conteudo do video e geracao de texto.

4. Armazenamento e retencao

Tokens e registros de publicacoes sao armazenados apenas localmente, na maquina do operador, e podem ser apagados manualmente a qualquer momento. Nenhum banco de dados remoto e mantido pelo Aplicativo.

5. Direitos do usuario

Como o Aplicativo opera sobre uma unica conta controlada pelo proprio operador, este pode a qualquer momento revogar o acesso do Aplicativo diretamente nas configuracoes de aplicativos conectados da sua conta TikTok, e apagar quaisquer dados locais armazenados.

6. Contato

Para questoes sobre esta politica, contate o operador do Aplicativo pelo e-mail associado a conta de desenvolvedor cadastrada no TikTok for Developers.
